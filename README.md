Music Store Analysis using SQL:

Overview:
This project involves analyzing the music store database using SQL queries to gain insights into customer behavior, sales trends, and product performance. The goal is to provide actionable recommendations for business stakeholders to improve sales and customer engagement.

Project Structure:
data/: Contains the music store database schema and sample data
queries/: Contains SQL queries used for analysis
results/: Contains output of SQL queries and visualizations
README.md: This file

Database Schema : The music store database consists of the following tables:
customers: Customer information
orders: Order details
products: Product information
order_items: Order item details
genres: Genre information.
artists: Artist information.

Results: The output of the SQL queries and visualizations are located in the results/ directory.

Tools Used:
SQL (PostgreSQL)
pgAdmin4
